# GSDetector

## Camera
Q6045 E MKII - PTZ Camera (60Hz)


## gsDetect Documentation

### Dependencies
1. Python 3.11.7
2. venv
3. Wave

### Activating the virtual environment
1. Navigate to gsDetect/Scripts/
if you are on Linux/MacOS
```
./activate
```
if you are on Windows
```
activate
```
### Using the AuxWave module
In main.py, write
```
import AuxWave
```
